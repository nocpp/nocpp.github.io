    # flex-grow 与 flex的区别

    1. 剩余宽度计算方式不同, flex-grow的剩余宽度等于容器宽度减去所有项目宽度和, 而flex的剩余宽度等于容器宽度减去设置了flex属性的项的宽度和
    2. 计算宽度方式都是剩余宽度*设置比例/设置了项目个数
    3. flex是几个值的简写，flex-grow, flex-basis, flex-shrink
